# Lancement rapide ATTRIO CAMPUS

## Structure du dossier

- Les lanceurs restent à la racine du dossier
- Tout le projet technique est rangé dans `Projet ATTRIO CAMPUS`

## App Mac

- Double-clique sur `ATTRIO CAMPUS.app`
- C’est le vrai lanceur Mac natif du projet
- Il ouvre Terminal, lance le serveur, puis ouvre le navigateur

## Mac

- Double-clique sur `Lancer ATTRIO CAMPUS.command`
- Le terminal s'ouvre, lance le serveur, puis ouvre le navigateur

## Windows

- Double-clique sur `Lancer ATTRIO CAMPUS.bat`
- La fenêtre de commande lance le serveur, puis ouvre le navigateur
- Pour générer un vrai `.exe` Windows avec l’icône Attrio, lance `Projet ATTRIO CAMPUS/scripts/build-windows-launcher.ps1` sur un PC Windows

## Raccourci navigateur

- Double-clique sur `Ouvrir ATTRIO CAMPUS.html`
- Il ouvre directement `http://127.0.0.1:5173`
- Le serveur doit déjà être lancé

## Icônes natives

- Icône Mac : `Projet ATTRIO CAMPUS/launchers/ATTRIO CAMPUS.icns`
- Icône Windows : `Projet ATTRIO CAMPUS/launchers/ATTRIO CAMPUS.ico`
